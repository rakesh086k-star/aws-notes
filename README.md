Dedicated Flex

If you buy 10 Windows 365 Flex licenses:
	•	You can provision up to 30 dedicated Cloud PCs (1:3 ratio).
	•	Each user gets their own personal Cloud PC.
	•	Only 10 users can be active at the same time.
	•	This works well for shift-based workers (Morning, Evening, Night).

Example:
	•	License 1 → User A (Morning), User B (Evening), User C (Night)
	•	All 3 have their own Cloud PC.
	•	But only one user uses that license at a time.

⸻

Shared Flex

If you buy 10 Windows 365 Flex licenses:
	•	You can assign access to more than 10 users (for example, 30 users).
	•	However, only 10 users can log in simultaneously.
	•	The 11th user must wait until another user signs out.
	•	Users share Cloud PCs instead of having their own dedicated Cloud PC.

Example:
	•	Total users = 30
	•	Licenses = 10
	•	Concurrent users allowed = 10
	•	If 10 users are already connected, the 11th user cannot connect until a session becomes available.
